# React, Social Login 프로젝트 
🚀 React와 Firebase를 이용하여 간단한 프로젝트 제작

## 개발 환경 🚩

* WebStorm
* TypeScript 4.9.5
* React 18.2.6

## 설치한 모듈 ⭐️

* react-router-dom
* react-bootstrap
* firebase

# 프로젝트 목표

1. page 구성 : home, event, login
2. event page는 로그인한 사용자만 접근
3. 인증 구현은 firebase 이용 ( provider 2개 이상 사용 )
4. 리액트 라우터 , context은 필수 사용
5. 배포는 https://vercel.com/
